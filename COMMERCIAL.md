# Commercial Licensing

This project is currently available under open-source licensing terms.

agentLIBRE may be offered under a separate commercial license in the future,
but no commercial license is currently offered or granted by this repository.

If commercial licensing is introduced later, it may be appropriate for users
who want commercial terms that differ from the LGPL, including proprietary
distribution, static linking, commercial support, warranty terms, indemnity,
access to official services, or permission to use official branding.

Any future commercial license would be available only under a separate written
agreement with the agentLIBRE maintainers.

Using the open-source license does not grant commercial-license rights.
If a commercial license is offered in the future, receiving one will not remove
rights already granted to the public under the open-source license for
open-source releases.

For now, the only public licensing path for this repository is the open-source
license stated in `LICENSE.md`.
