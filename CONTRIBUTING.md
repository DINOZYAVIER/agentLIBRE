# Contributing

agentLIBRE is a personal project at this stage.

## External contributions

External code contributions are not being accepted yet.

Please do not open pull requests with code, documentation, assets, prompts, or
other project material unless the maintainer has explicitly asked you to do so
for a specific issue.

## Issues

You may open GitHub issues for:

- bug reports;
- reproducible failures;
- feature requests;
- design feedback;
- documentation problems;
- questions about expected behaviour.

Do not include third-party code or large patches in issues unless requested.
Small excerpts that are necessary to explain a bug are fine.

## Future contribution policy

If the project starts accepting external contributions later, the maintainer
will publish an explicit contribution policy before accepting non-trivial
patches.
